# Mate Selection

This package is a collection of mate-selection strategies for evolutionary algorithms.

Mate selection algorithms randomly select pairs of individuals from a
population, with the intent of mating them together. Individual are selected
based on their reproductive fitness "score".

* [**docs.rs**](https://docs.rs/mate_selection)
